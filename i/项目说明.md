# 项目说明

## 概述
这是一个基于 Vercel + PHP 的 302 重定向代理项目。通过 Vercel 的 Serverless PHP 运行时，将自定义域名的所有请求重定向到指定的目标服务器 IP:Port。

## 核心功能
- 接收用户请求，获取完整的 URI（路径 + 查询参数）
- 读取环境变量中的目标服务器 IP、端口、协议
- 执行 302 临时重定向到目标服务器
- 保留原始请求的路径和查询参数

## 部署方式
- 平台：Vercel
- 运行时：PHP（vercel-php@0.7.1）
- 配置文件：vercel.json
