# 项目技术细节

## PHP 脚本逻辑（index.php）

### 环境变量读取
- `TARGET_IP`：目标服务器 IP（必填，为空则返回 500）
- `TARGET_PORT`：目标端口（可选，默认 80）
- `TARGET_SCHEME`：协议（可选，默认 http）

### 请求处理
1. 获取 `$_SERVER['REQUEST_URI']` 获取完整路径+查询参数
2. 构建目标 URL：`{scheme}://{ip}:{port}{uri}`
3. 非 80/443 端口才拼接端口号
4. 通过 `header()` 发送 302 重定向

### 示例
- 访问 `https://你的域名/docs/1?sort=asc`
- 跳转到 `http://服务器IP:8080/docs/1?sort=asc`

## Vercel 配置（vercel.json）

### functions
- 指定 `index.php` 使用 `vercel-php@0.7.1` 运行时

### rewrites
- 将所有路由 `/(.*)` 重写到 `/index.php`
- 确保 SPA 风格的路由全部由 PHP 处理

## 部署步骤
1. 创建项目文件夹，放入 `index.php` 和 `vercel.json`
2. 推送到 Git 仓库（GitHub/GitLab/Gitee）
3. 在 Vercel 导入项目
4. 设置环境变量：TARGET_IP、TARGET_PORT、TARGET_SCHEME
5. 部署并绑定域名
