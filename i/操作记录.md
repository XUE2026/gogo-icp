# 操作记录

## 2026-06-23

### 创建 PHP 跳转代理项目
- 创建 `index.php`：核心 PHP 跳转脚本，读取环境变量 `TARGET_IP`、`TARGET_PORT`、`TARGET_SCHEME`，将请求 302 重定向到目标服务器
- 创建 `vercel.json`：Vercel 部署配置文件，使用 `vercel-php@0.7.1` 运行时，所有请求重写到 `index.php`

### 项目目的
通过 Vercel 部署 PHP 反向代理，将自定义域名的请求 302 重定向到真实的服务器 IP:Port，隐藏真实服务器地址。

### 环境变量
| 变量名 | 说明 |
|--------|------|
| TARGET_IP | 目标服务器公网 IP（必填） |
| TARGET_PORT | 目标端口（默认 80） |
| TARGET_SCHEME | 协议（默认 http） |
